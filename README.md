# Vault Dashboard

A modern, responsive fintech dashboard built with React, Vite, and Tailwind CSS. Features a sleek dark/light mode theme with neon accents and comprehensive banking functionality.

## 🚀 Features

### Core Functionality
- **My Bank** - Unified cards and bank account management
  - Virtual card display with 3D design
  - Link external bank accounts
  - Card settings and controls
- **Overview** - Financial dashboard with net worth, spending, and activity
- **Transactions** - Searchable transaction history with filters
- **Loans** - Loan tracking and management
- **Investments** - Crypto & stock portfolio
- **Bill Pay** - Bill management with crypto payment option
- **Settings** - Profile, notifications, security, and language preferences

### User Experience
- **Authentication** - Sign in/Sign up pages with glassmorphism design
- **Dark/Light Mode** - Toggleable theme with localStorage persistence
- **Responsive Design** - Mobile-first approach with sidebar (desktop) and bottom nav (mobile)
- **Modern UI** - Neon accents, smooth animations, and premium aesthetics

## 🛠️ Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Client-side routing
- **Lucide React** - Icon library
- **CSS Variables** - Dynamic theming

## 📦 Installation

```bash
# Clone the repository
git clone <your-repo-url>
cd vault-dashboard

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🔒 Security Note

This is a **demo application** for UI/UX purposes. It does not handle real financial data or credentials. All authentication and banking features are mockups.
